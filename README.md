# 🍃 Hazel AI | Full-Stack Intelligent Assistant

Hazel is a professional-grade AI chatbot built with a **Node.js** backend and a **Glassmorphism** frontend. It features secure API handling, responsive design, and real-time communication.

## 🚀 Features
- **Secure Architecture:** API keys are shielded via server-side proxy.
- **Modern UI:** Built with CSS Grid, Flexbox, and Frosted Glass aesthetics.
- **Full-Stack:** Integrated with a Node.js/Express backend.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **Backend:** Node.js, Express.js, Axios
- **Environment:** Dotenv for secret management

## 📦 Installation
1. Clone the repo: `git clone https://github.com/your-username/hazel-ai.git`
2. Install dependencies: `npm install`
3. Create a `.env` file and add your `HAZEL_API_KEY`.
4. Run the server: `npm start`
